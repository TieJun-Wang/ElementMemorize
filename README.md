# ElementMemorize
元素周期表背诵练习小程序
一款纯 Python 命令行工具，专为化学学习者打造，通过交互式问答高效记忆元素周期表核心知识点。
**功能介绍**
多种练习模式
序号→中文名
中文名→序号
序号→符号
符号→序号
符号→中文名
原子半径排序比较
电负性排序比较（待完善）
自定义练习范围：1–118 号元素自由选择
难度分级：普通 / 中等 / 困难 / 地狱 / 自定义题量
三次答题机会，答错显示正确答案
实时统计：正确率、总用时、平均答题时间
错题自动记录，便于针对性复习
输入格式校验：中文名、元素符号自动规范
**技术栈**
语言：Python
依赖：json、random、time、re
数据：本地元素信息 JSON 文件
使用方法
准备 elements_info.json 元素数据文件
运行 Periodic Table.py
按菜单选择练习模式与范围开始练习
